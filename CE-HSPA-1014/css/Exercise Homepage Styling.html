Now it's time to add some styling to your homepage. 

Remember: You know a lot of styling properties now, which means you can make something that's really beautiful... or really hideous. With great power comes great responsibility.

Overview:

Add some styling that you think is visually pleasing to your homepage. 

Experiment with some of the properties that we've looked at, including:

  • text
    • color
    • font-size
    • font-weight
    • font-style
    • font-family
    • font-variant
    • text-align
    • text-decoration
    • text-shadow
  • others
    • background-color
    • padding
    • margin
    • width
    • height
    • display

Process:

  • Create a new file named styles.css
  • Add a link tag to the top of your homepage index.html file to connect the HTML to the CSS:
    <link rel="stylesheet" href="styles.css" />
  • Add some styles to your CSS file
  • Save both HTML and CSS files
  • Open your file in a browser to view and test the homepage styling
  • Continue editing until you're satisfied with the styling
